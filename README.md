# Introduction to Cypress

## What is Cypress?
Itâ€™s a test automation tool, similar to Selenium WebDriver, WebDriverIO, TestCafe, and others.

## Setting up
Download the project and type on terminal inside the project folder

`npm install`

## Running the tests
To run a test, type on terminal
 
 `npx cypress open` 
 
 or 
 
 `npx cypress run`, if you want to run without the browser
